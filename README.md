# Neil's Second Dinner Technical Producer Application
You can play the game directly in a desktop web browser (tested in both Chrome and Firefox): <br>[Neil's Technical Producer Application Game](https://www.neilmehta.net/seconddinner)

## Setup
### Requirements
- Unity 6000.3.6f1 or higher

The project is composed of 2 main scenes:
- `Assets/_Game/Scenes/Start Scene.unity` - The default starting scene. Can be skipped if you want. Provides the game lore and then transitions to the `Game Scene` on button click
- `Assets/_Game/Scenes/Game Scene.unity` - Main gameplay experience. See the [Trello board](https://trello.com/b/dZVRdZSE/neils-second-dinner-application-game) for a full breakdown of features.

## Technical Details
This project was built over 2 days to demonstrate my comfort with Unity, Github, and Trello. 

I used the following AI tools
- Cursor (with Unity MCP) to create the initial game state machine
- Gemini (Nano Banana) to generate all textures used in the game, since I'm not a strong artist
